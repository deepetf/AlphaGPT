# 可转债因子挖掘稳健性增强计划

## 目标
在不改变模型结构的前提下，通过回测评估与训练奖励改造，提升因子在不同时间段与市场状态下的稳定性与可交易性。

## 计划分解

### 1. 分段验证奖励
- 将回测区间划分为训练段、验证段、近端段。
- Reward 采用分段加权平均，避免单段拟合。
- 若跨段收益或 Sharpe 方向不一致，施加惩罚。

### 2. 加入稳定性项
- 计算滚动 Sharpe 或滚动 IC 的波动度。
- Reward 中扣除不稳定性惩罚（波动越大惩罚越强）。

### 3. 增加回撤惩罚
- 计算最大回撤与回撤持续期。
- Reward 中显式惩罚高回撤因子，降低极端路径依赖。

### 4. 活跃度与可交易性约束
- 若有效持仓数不足或流动性低于阈值，增加惩罚。
- 引入强赎风险或剩余期限过短的过滤/惩罚项。

### 5. 轻量长度惩罚
- 保留公式长度惩罚，但降低权重。
- 让稳健性与可交易性指标成为主导信号。

## 初步落地点
- model_core/backtest.py: 增加分段评估、滚动稳定性、回撤指标、活跃度惩罚。
- model_core/engine.py: 调整 reward 汇总方式，弱化长度惩罚权重。
- model_core/config.py: 增加稳健性相关配置项（权重、分段比例、阈值）。

## 输出与验证
- 训练日志输出分段收益、Sharpe、稳定性指标与回撤。
- 产出文件中记录稳健性指标，便于筛选与对比。

## 🔍 差异分析

### 差异来源：整手交易约束

**回测**：
- 理想化等权分配
- 每个标的权重精确 = 1/K = 10%

**模拟**：
- 整手交易（10张为单位）
- 向下取整导致实际权重 < 10%
- 资金利用率 < 100%

**示例**：
```
初始资金: 100,000元
目标持仓: 10只
理想分配: 10,000元/只

标的A (价格100元): 可买100张 → 投入10,000元 ✅
标的B (价格120元): 可买80张  → 投入9,600元  ❌ (少400元)
标的C (价格95元):  可买100张 → 投入9,500元  ❌ (少500元)
```

### 为什么接受这个差异？

1. **这是实盘的真实约束**，而非逻辑错误
2. 选股一致性100%，证明核心逻辑无误
3. 0.90的相关性已经很高，足以证明策略可靠性
4. 实盘必然存在整手约束，无法做到理想化等权
